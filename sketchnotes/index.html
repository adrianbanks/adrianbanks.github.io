Text will match in any of the meta-data of the sketchnote (title, speaker, event, or tags).

To search within a specific field, use the relevant prefix:

  • title - searches only within the title (e.g. title:plain)
  • speaker - searches only within the speaker (e.g. speaker:dylan)
  • conference - searches only within the conference (e.g. conference:"NDC London")
  • event - searches only within the event (e.g. event:"NDC London 2022")
  • tag - searches only within the tags (e.g. tag:legacy-code)
